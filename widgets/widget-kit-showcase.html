Designer notes

  • 1 · Match the Words — Upgrades the current static "vocabulary list + fill-in-blank" review with a tactile drag-to-match. Optional non-graded timer adds replay motivation without pressure.
  • 2 · Memory Match — Replaces flashcard slideshows with an active concentration game; doubles as a culture touchpoint (Roman gods).
  • 3 · Sort the Words — Upgrades multiple-choice "which is Latin?" items into a hands-on two-bin sort that makes word-origin thinking visible.
  • 4 · Guess Where It Came From — New "Living Latin" wonder moment; replaces passive did-you-know text boxes with a guess-then-reveal that rewards curiosity. Etymologies fact-checked (octō → October; octopus noted as Greek sharing the same ancient root for eight).
  • 5 · Magister Caesar — New recurring guide persona; replaces plain instruction text with a friendly read-aloud character for early/emerging readers.
  • Deployment: All widgets are self-contained HTML/CSS/JS intended to be hosted on GitHub Pages and embedded via <iframe> into Canvas lesson pages, per the canvas-interactive-widgets workflow. No external JS libraries; only the Optima owl logo loads remotely.
