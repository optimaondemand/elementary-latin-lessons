ELEMENTARY LATIN — WIDGET LIBRARY DEMO & SPEC

Data-driven widget library

Link elatin-widgets.css and elatin-widgets.js in any lesson page, then drop in one of the seven declarative elements below. The script auto-scans on load and turns each element into a working, mouse+touch game with its own Reset. Config lives in data-* attributes: wrap the attribute in single quotes so the JSON inside can keep its double quotes. This page shows one live example of each widget and doubles as the authoring spec.

1 · Drag-Match  ·  .elw-match

data-title heading · data-pairs array of [Latin, meaning] · data-timer "true" adds the optional, never-graded "beat your best time" toggle (localStorage best time, off by default).

<div class="elw-match" data-title='Match the words' data-pairs='[["salvē","hello"],["valē","goodbye"],["grātiās","thank you"]]' data-timer="true"></div>
2 · Flip Memory  ·  .elw-memory

data-title heading · data-pairs array of [cardA, cardB]; each pair becomes two cards that match each other. Emoji are welcome in either side.

<div class="elw-memory" data-title='Memory match' data-pairs='[["Neptūnus","🔱 sea"],["Iuppiter","⚡ sky"]]'></div>
3 · Sort Bins  ·  .elw-sort

data-title heading · data-bins array of bin labels (any number) · data-items array of [label, binIndex] where binIndex points into data-bins.

<div class="elw-sort" data-title='Sort the words' data-bins='["Comes from Latin","Not from Latin"]' data-items='[["aquarium",0],["ninja",1]]'></div>
4 · Guess / Living Latin  ·  .elw-guess

data-title heading · data-rounds array of round objects: icon (emoji), prompt, options (array), answer (index of the correct option), reveal (the ✨ explanation shown after any choice). Multiple rounds get a Next ▶ button.

<div class="elw-guess" data-title='Where did it come from?' data-rounds='[{"icon":"🚀","prompt":"The NASA program Artemis…", "options":["Mārs","aqua","octō"],"answer":0, "reveal":"✨ Mars rovers are named for the Latin war-god Mārs!"}]'></div>
5 · Talking Guide  ·  .elw-caesar

data-lines array of lines the guide reads aloud (▶ Play uses speechSynthesis and cycles through them) · data-name optional guide name (default "Magister Caesar"). Falls back gracefully when browser speech is unavailable.

<div class="elw-caesar" data-lines='["Salvē, young scholar!","Today we meet the numbers of Rome."]'></div>
6 · Explore  ·  .elw-explore

data-title heading · data-cards array of card objects: emoji, name (Latin), gloss (English), fact (a short, age-appropriate fact). Renders tappable cards; tap flips the card to reveal its fact, tap again to close. Reset closes them all. Real buttons + aria-expanded → click, touch and keyboard.

<div class="elw-explore" data-title='Meet the Gods of Rome' data-cards='[{"emoji":"⚡","name":"Iuppiter","gloss":"Jupiter", "fact":"King of the sky. His thunderbolt was a sign of power."}]'></div>
7 · Self-Check  ·  .elw-selfcheck

data-title heading (default "Show what you remember") · data-questions array of question objects: q (the prompt), options (array of choice strings), answer (0-based index of the correct option). Ungraded practice — correct locks green with a "Yes!"; wrong nudges amber and lets the student retry. Per-widget Reset.

<div class="elw-selfcheck" data-title='Show what you remember' data-questions='[{"q":"Which god ruled the sea?", "options":["Iuppiter","Neptūnus","Iūnō"],"answer":1}]'></div>
